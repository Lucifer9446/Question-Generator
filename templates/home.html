Welcome to Question Generator!

Question Generator is an online tool that automatically generates questions based on provided text.
Whether you're a student preparing for an exam or a teacher looking to create practice materials,
Question Generator can save your time and effort by generating a variety of question types.


With Question Generator, you can generate:

  • Multiple Choice Questions (MCQs): Each question comes with multiple options, and you need to select the correct answer.
  • True/False Questions: Each question requires you to determine if a given statement is true or false.
  • Short Answer Questions: Each question prompts you to provide a brief answer based on the context.

Getting started is easy. Simply enter the text you want to generate questions from, select the question type and the number of questions you want to generate, and click on the "Generate Questions" button. The generated questions will be displayed on the next page.

Start using Question Generator today and make the process of generating questions quick and hassle-free!
